# Silhouettes (Fabric 1.21.5)

Client-only mod that spawns distant black silhouettes with red eyes and a spawn sound. Configure via `config/silhouettes.properties`.

Build & CI

1. Create a GitHub repo and push this folder.
2. GitHub Actions will run the build and upload artifact (see `.github/workflows/build.yml`).

Local build (requires Java 17):
```
./gradlew build
```
